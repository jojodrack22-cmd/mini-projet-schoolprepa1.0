# SchoolPrepar

Plateforme d'orientation scolaire pour aider les élèves à choisir leur parcours académique.

## Technologies

- Symfony
- Twig
- PHP
- MySQL

# Installation

git clone projet

composer install

symfony server:start

# Fonctionnalités

- Consultation des filières
- Recherche d'établissements
- Orientation personnalisée
